# 3D Dice Roller

A fully functional 3D dice simulator built with HTML, CSS, and JavaScript.

## Features
- Realistic 3D dice with all 6 faces
- Smooth rolling animation
- Random dice roll results
- Responsive design
- Keyboard support (Space/Enter to roll)

## How to Use
1. Open `index.html` in your browser
2. Click the "Roll Dice" button or press Space/Enter
3. Watch the dice roll and see the result

## Technologies Used
- HTML5 for structure
- CSS3 for 3D transforms and animations
- Vanilla JavaScript for interactivity
